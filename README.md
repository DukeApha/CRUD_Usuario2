# CRUD de Usuarios con Git Flow

Proyecto desarrollado como parte de la asignatura **Programación III**, donde se implementa un CRUD funcional en C# (WinForms + SQL Server) aplicando correctamente la metodología **Git Flow**.

---

## 🎯 Objetivo

Demostrar el uso correcto de:
- Control de versiones con Git
- Flujo de trabajo Git Flow
- Manejo de ramas (feature, dev, qa, main)
- Pull Requests (PR)

---

## 🛠️ Tecnologías utilizadas

- C# (WinForms)
- SQL Server
- Git & GitHub

---

## ⚙️ Funcionalidades del CRUD

- ✔ Crear usuario  
- ✔ Listar usuarios  
- ✔ Actualizar usuario  
- ✔ Eliminar usuario  

---

## 🌿 Flujo de trabajo (Git Flow)

Se implementó el siguiente flujo:


Cada feature generó:
- 1 PR hacia `dev`
- 1 PR hacia `qa`
- 1 PR hacia `main`

Total: **15 Pull Requests**
